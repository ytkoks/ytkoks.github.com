CSSを使ってページ全体を中央寄せにする設定は、以下のようにします。

HTML: 

<div id="container">
    <p>contents</p>
</div>

CSS: 

body {
    text-align: center;
}
div#container { 
    margin: 0 auto;
    width: 800px;
}
